# Hospital-API
An host of APIs for a hospital usage.

# Dependencies

Express <br />
Mongoose <br />
Body-parser <br />
Passport <br />
JWT <br />
Passport-JWT <br />
Mocha <br />
Chai <br />
Should <br />

Entry Point: index.js

Instructions to run:

1. First run "mongod" to fire-up the Db.
2. Then run "node index.js" for fire-up the server.
3. Use Postman or any other platform to use for front-end.
